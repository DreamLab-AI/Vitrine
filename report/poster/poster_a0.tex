\block{The Challenge}{
\textbf{Cultural heritage is fragile, immovable, and under-documented.} Galleries,
museums and historic interiors are typically captured as flat photography or
sparse photogrammetry that loses material appearance, view-dependent lighting and
the spatial relationships between objects.

This programme trials a production pipeline that turns a \textbf{single hand-held
4K video walkthrough} into an explorable, photoreal \emph{and} structured 3D scene
— a volumetric record suitable for archival, remote study and immersive display.
\vspace{4mm}

\begin{tikzfigure}[Source capture: a single 4K walkthrough of a gallery interior.]
\includegraphics[width=0.96\linewidth]{src_frame.jpg}
\end{tikzfigure}
}

\block{Hierarchical Scene Graph}{
The pipeline's central contribution is moving from a \emph{monolithic} radiance
field to a \textbf{structured, editable scene}. SAM2 promptable segmentation
decomposes the capture; masks are projected from 2D into the Gaussian field with
\textbf{98.3\,\% coverage}, resolving \textbf{37 detections into 31 clean object
prims}.
\vspace{4mm}

polygonal twin without re-authoring the scene.

\block{System Architecture}{
\textbf{Three coordinated GPU containers} run the heterogeneous toolchain:
\vspace{2mm}

\begin{itemize}\setlength{\itemsep}{2mm}
\item \textbf{gaussian-toolkit} — CUDA 12.8, capture$\rightarrow$splat$\rightarrow$TSDF
\item \textbf{milo} — CUDA 11.8, MILo \& GaussianWrapping meshing
\item \textbf{come} — CUDA 12.1, CoMe manifold meshing
\end{itemize}
\vspace{2mm}

Built on \textbf{LichtFeld Studio v0.5.2} with a native USD I/O path and an
\textbf{MCP control server} (70+ tools) for agent orchestration. Hardware: dual
\textbf{RTX 6000 Ada} (96\,GB VRAM), Threadripper PRO, 251\,GB RAM.
\vspace{3mm}

\end{tikzfigure}
